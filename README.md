# agendamento-api

API REST para Agendamento de Consultas do e-SUS APS / PEC.

## Pre-requisitos

- Java 21
- Maven 3.9+

## Build

```bash
mvn clean package
```

## Execucao

```bash
java -jar target/agendamento-api.jar
```

## Variaveis de ambiente

- `DATABASE_URL`
- `DATABASE_USERNAME`
- `DATABASE_PASSWORD`

## Swagger

- http://localhost:8080/swagger-ui.html
